Introduction

HTML et CSS sont deux langages informatiques utilisés par les technologies du Web. L'HTML ou HyperText Markup Language va décrire la structure d'un document et ses relations avec d'autres documents (HTML mais aussi des images, vidéos...) à l'aide de balises. Le CSS pour Cascading Style Sheet va quant à lui décrire des règles d'affichage pour un document HTML, comme la taille du texte ou sa couleur pour ne citer qu'elles.

Les balises vont former des blocs emboités les uns dans les autres qui contiennent le contenu du document, permettant une structuration hiérarchisée des éléments.

Les balises

Il existe différents types de balises : tout d'abord des balises structurantes comme header (en-tête), footer (pied de page), p (paragraphe)... Il existe aussi la balise img pour insérer une image, audio pour du son ou encore a pour créer un hyperlien.

Ces balises fonctionnent soit par pairs (balise ouvrante et balise fermante) comme header : <header></header>, soit toutes seules, comme par exemple img : <img>.

Il est aussi possible de créer des listes avec ul et des balises enfantes li:

  • Un élément de la liste
  • Un deuxième élément
    • Une sous-liste

Une autre balise notable est table qui permet de créer des tableaux :

La mise en forme

Pour mettre en forme le document ainsi créé, l'utilisation de CSS devient primordiale. Pour chaque élément HTML, CSS nous offre la possibilité de modifier des propriétés. Ces propriétés sont très variées, elles concernent la taille et le positionnement des blocs, la couleur de fond, l'affichage du texte... elles permettent de modifier quasiment tout le comportement du navigateur quand celui-ci va afficher le document.

Pour cibler l'élément sur lequel on veut appliquer des propriétés, on utilise un sélecteur CSS. Un sélecteur CSS est un ensemble de critères qui permet au navigateur de trouver les éléments correspondants. On peut par exemple modifier la couleur de tous les liens avec le code CSS suivant :

a { color: red; }

On sélectionne tous les éléments a et on leur modifie la propriété color qui controle la couleur du texte.
